* **Python 3.7+** installed.

  ```bash
  python --version
  ```
* **pip** (comes with modern Python).
  Check with:

  ```bash
  pip --version
  ```
* **Git** (optional, for cloning).
  Check with:

  ```bash
  git --version
  ```
